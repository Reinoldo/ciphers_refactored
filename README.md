Ciphers_Refactored
==================

Componente de cifras, que implementa a criptografia dos algoritmos de cifra de césar, ROT13, ROT47 e Vigenere.
Este é exercício é proposto no tutorial "Coding By Example", que foi apresentados em eventos como AgileBrazil 2012 (São Paulo), 
Agiles 2012 (Córdoba/Argentina) e CBSoft 2013 (Brasília).

Mais informações em www.codingbyexample.org
